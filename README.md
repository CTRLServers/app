# CTRLServers APP
Hello! Welcome to the repository. This is an code from CTRLServers App, here is what it includes:
- Basically working app
- SFTP Tab
- Adding pterodactyl, VPS or VDS servers, managing them (fully working control panel for servers)
- Keychain tab (in case if VPS or VDS doesn't accept passwords)
Andd much more i think?
Developed on Node JS, Electron. Used HTML + CSS + JS. May switch to framework soon.
What you can do with this project? You can use part of the code in your projects, BUT provide that this part of code was made by CTRLServers, you can do forks of this project, you can download this code and edit it how you like, and etc.
May include shit code, sorry for that.

How to run this? Just execute npm install and after that npm start.
How to build this? Use npm run build:os, for example npm run build:win.
